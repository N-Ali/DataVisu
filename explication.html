Description du projet

Pour le cours D'Analyse, Gestion et visualisation des données massives, nous avons choisi de représenter les données d'une base de données qui concerne les différents Animés Japonais

Processus

Tout d'abord, nous avons choisi notre base de données sur Kaggle qui est une plateforme web qui contient de nombreuses bases de données. Une fois notre base de données choisi, nous nous sommes aperçu qu'il été difficile d'exploiter ces données, nous avons donc optée pour une seconde base de données, qui celle-ci correspondait mieux à notre besoin. Dans un premier temps, une fois la base de données choisis nous avons fait un tri, effectivement, la base de données contenait 12 000 lignes de données, nous l'avons réduit jusqu'à avoir 500 données. En outre, nous avons éliminé les données non exploitables ou inintéressantes tel que le titre de l'oeuvre dans différentes langues. Enfin, une fois que nous avions recueillis toutes les données que nous voulions, nous avons commencé à implémenter les différents graphes pour visualiser les données choisis.

Nous avons donc choisis de montrer les données suivantes:

  • Titre
  • Images
  • Notes
  • Sources
  • Studio
  • Score
  • Genre
  • Date de Diffusion

    • Analyse des diagrammes


    Premier diagramme

    La première visualisation qui est la cardboard permet de visualiser toutes les données disponible sous forme de card, il est possible de filtrer les données via leur genre, de plus à l'aide de la barre de recherche il est possible de rechercher un animé par son titre, mais aussi par le studio de productions qui listera tout les animés produit par le studio en question .

    Le premier diagramme nous montrent les différents Types d'animés que compose notre base de données, il suffira à l'aide de la souris de rester sur l'élément choisi pour avoir les chiffres exactes de chaque genres. En effet, on peu remarquer que dans notre base de données, 281 animés sont des séries TV, 87 sont des OVA(Original Video Animation), et 121 sont des films.


    Deuxième diagramme

    Le deuxième diagramme, quant à lui nous montre les diffusions des animés, à l'instar du premier diagramme, pour avoir les différents chiffres des diffusion il faudra rester sur l'élément, ainsi on peu voir qu'il y'a 3 types de diffusion :

  • "Diffusion en cours", qui correspond aux animés qui sont diffusé à l'heure actuelle
  • "Diffusion terminé" qui correspond aux animés terminés et qui ne sont plus en productions
  • "Pas encore diffusé" qui, comme son nom l'indique, correpond aux animés qui ne sont pas encore diffusé.
    • Ainsi, on peu remarquer que 477 des animés present sont terminés, tandis que 11 sont encore en diffusion et qu'1 n'est pas encore diffusé.


    Troisième diagramme

    Le troisième et dernier diagramme circulaire nous montre le rating des animés, le rating correspond à l'age conseillé pour les visionnés. On remarque qu'il y'a 6 rating:

  • PG-13 Adolescent ou plus avec un nombre de 284
  • G Tout-Age avec un nombre de 47
  • R+ Nudité douce avec un nombre de 53
  • R - 17+ Violence et blasphèmes avec un nombre de 43
  • PG-Enfants avec un nombre de 60
  • Rx Hentai avec un nombre de 2
    • On peu constater que dans un panel de 500 animés, la moitié est destiné pour les adolescents de 13 minimum


    Quatrième diagramme

    Ce diagramme est un diagramme à bar,en effet l'axe des abscisses indique les différents studios tandis que l'axe des ordonnées nous indique le nombre d'oeuvre que le studio à produits. On peu voir que le studio Toei Animation est le studio qui a produit le plus d'animés. Toei Animation est un voir le plus gros studio animation du Japon
